# Rese（リーズ）
概要説明
・飲食店予約サービス
・会員登録して、飲食店の予約ができる
・お気に入り登録も可能　

## 作成した目的
模擬案件の為。

## 機能一覧
- 会員登録
- ログイン
- ログアウト
- ユーザー情報取得
- ユーザー飲食店お気に入り一覧取得
- ユーザー飲食店予約情報取得
- 飲食店一覧取得
- 飲食店詳細取得
- 飲食店お気に入り追加
- 飲食店お気に入り削除
- 飲食店予約情報追加
- 飲食店予約情報削除
- エリアで検索する
- ジャンルで検索する
- 店名で検索する

## 使用技術（実行環境）
- Laravel　8.83.23
